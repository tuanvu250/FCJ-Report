### Week 3 Objectives:

- **Advanced Storage**: Master **Amazon S3** (Classes, Lifecycle, Access Points, Glacier) and security features.
- **Data Migration**: Understand data migration solutions with **Snow Family** and **Storage Gateway**.
- **VM Import/Export**: Practice migrating On-premise VMs to AWS and vice versa with **VM Import/Export**.
- **Resource Optimization**: Optimize EC2 costs using **Lambda** and manage resources efficiently with **Tags/Resource Groups**.
- **Backup Strategy**: Deploy a comprehensive backup strategy for the system with **AWS Backup**.
